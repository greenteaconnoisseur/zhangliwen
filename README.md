## 前言
人生就像小说一样，总是意料之外，却情理之中。我从来没想过，我对她的信任会崩塌。

我和她好了两年，在我的心目中，她柔弱、单纯并善良。我从未怀疑过她说的任何话，我也无论如何也无法相信，她是一个会骗人的女孩。两年间，我与她像家人一样生活，我认为我们真的会走到最后，然而，事情却猝不及防的朝着截然相反的方向发展。

这半年时间，我一直在为她父亲的去世感到难过。我已经把当她亲人，心里想的是怎样同她一起度过难关。当她折磨我的时候，我会告诉自己，她经历了这么大的变故，一定更难过，因此我要包容她。令人意想不到的是，在这种情况下，她还可以进行劈腿操作…？这让我不禁怀疑，她口中的一切事情的发生，都是真实的吗？

她可以在父亲去世的不久后劈腿，开启一段新的美美的恋情。
她可以利用他父亲的去世，让我对她无底线的忍让。
她可以在劈腿的时候和我一起旅行，会开心的拍照，会亲切的叫老公，然后回国立刻分手。
她可以将肆无忌惮的折磨作为对我全心全意付出的回应。
她可以伪装两年（甚至更久？）。

我从未遇到、也从未听说过如此生性凉薄之人。
